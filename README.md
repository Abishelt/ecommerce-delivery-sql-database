# E-commerce & Delivery SQL Database (SQLite)

## Overview
A relational database designed for a multi-vendor e-commerce and delivery platform, covering customers, orders, payments, shipments, loyalty, and inventory.

- **Database:** SQLite  
- **Tables:** 22  
- **SQL Focus:** Schema design + business reporting queries

## Repository structure
- `/sql/`  
  - `database_schema.sql` (tables + keys + constraints)
  - `sample_data.sql` (demo data)
  - `business_query.sql` (business reporting queries)
- `/docs/`
  - `Database_Structure.pdf` (schema documentation)
  - `ecomerce.pdf` or `ecomerce.md` (project report)
- `/db/` 
  - `E_Comerce.db`

## How to run (SQLite)
1. Create a new SQLite database
2. Run `/sql/database_schema.sql`
3. Run `/sql/sample_data.sql`
4. Run `/sql/business_query.sql`

## Example business questions answered
- Revenue and order trends
- Top vendors/products
- Customer behaviour and loyalty activity
- Delivery and fulfilment tracking
